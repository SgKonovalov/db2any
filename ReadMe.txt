При написании приложения, использованы:
1.	Java 6;
2.	Spring Framework (MVC, Data);
3.	ORM Hibernate;
4.	Maven;
5.	Tomcat;
6.	БД Derby;
7. 	Отправлно на GitHub с помощью Git.

Доступ к функционалу, осуществляется по запросу: http://localhost:8080/db2any/bykey/getjson/key=1

Web-приложение, состоящее из:
1.	Пакета ru.ucs.exe.Controller, включающего класс TtableController.
Настоящий класс осуществляет отправку ответа Клиенту в формате JSON.
2.	Пакета ru.ucs.exe.DAO, включающего 2 интерфейса и 2 класса, реализующий слой доступа к БД
	(подробное описание классов и интерфейсов в коде);
3.	Пакета ru.ucs.exe.Model, включающего класс Answer, являющегося моделью данных,
	предоставления ответа в формате JSON.
4.	Файла: META-INF/dataSource/dbConnection.xml – содержит все beans для подключения и работы с БД;
5.	Файла: META-INF/dataSource/jdbc.properties – содержит все данные для подключения и авторизации к БД;
6.	Файла: WEB-INF/servlets/appServlets/appServ.xml – конфигурация JSON конвертора;
7.	Файла: WEB-INF/servlets/root.xml – корневой сервлет приложения;
8.	Файла: WEB-INF/web.xml – дескриптор развёртывания;
9.	Файла: pom.xml – использованные библиотеки.
Более подробное описание функционала и используемых библиотеках/иные данные в комментариях к коду,
в самой программе.
